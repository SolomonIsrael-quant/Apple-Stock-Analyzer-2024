**Title:** Apple Stock Analyzer 2024

**Description:**
A Python-based Apple stock analyzer using Pandas, NumPy and yfinance. Downloads and analyzes real Apple (AAPL) stock data for the full 2024 trading year.

**Features:**
- Downloads real Apple stock data using yfinance
- Calculates daily returns using pct_change
- Generates 50-day and 200-day moving averages
- Produces Golden Cross buy/sell signals
- Identifies best and worst trading days
- Prints full year summary report

**Results:**
- Total trading days: 251
- Start price: $183.73
- End price: $250.83
- Total return: 36.52%
- Best day: June 11 2024 — +7.26%
- Worst day: August 5 2024 — -4.82%

**Libraries used:**
- Pandas
- NumPy
- yfinance

**How to run:**
1. Install libraries: pip install pandas numpy yfinance
2. Run apple-stock-analyzer.py
